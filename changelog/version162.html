Enjoy new features in Novemeber 2022 update!☄️🚀⚡

Novemeber update gives you more features & capabilities, privacy settings updates and bug fixes. Here you can watch list of changes in Novemeber 2022 update aka version 1.6.2.


Clock widget was updated

• The ability to connect multiple time zones has been added. Now you can look at the time in any city or region.

• Introducing the Analog Clock Widget. Now the clock is available in a new design and you can add it to the widget panel, it also supports location selection.

• Bug fixes & stability improvements.



Outlook widget improvements

• A bug with updating mail and contacts has been fixed.

• A bug with signing into a Microsoft account has been fixed.



Edge widget improvements

• Pinned sites will now sync.

• You no longer need to insert a prefix to search in a specific search engine.



Sticky notes feature was updated

• The graphical interface of notes has been seriously redesigned in favor of convenience, beauty and practicality.

• Notes search has been improved.

• Bug fixes & stability improvements.
